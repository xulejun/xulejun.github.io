完整图文配置Java环境变量

  • 右击桌面图标“此电脑”
  • 选择“管理”，出现以下界面
  • 选择界面上高级系统设置
  • 选择界面的“环境变量”
  • 点击“新建”
  • 1.配置PATH路径*
  • 变量名：“PATH”
  • 变量值选择“浏览目录”；
  • 选择安装jdk下的bin目录
  • 2.配置JAVA_HOME*
  • 变量名：“JAVA_HOME”
  • 变量值选择“浏览目录”；
  • 选择安装的jdk目录
  • 3.配置CLASSPATH*
  • 变量名：“CLASSPATH”
  • 变量值：“.;E:jdk-10.0.2;”
  • 路径为java下的jdk所有文件目录（将路径更改，格式为“.;路径;”）
  • 4.检查配置*
  • 快捷键windows/Command + R，输入cmd；
  • 进入dos窗口，输入javac，回车出现以下画面；
  • 输入java -version，回车出现以下画面
  • 恭喜，配置成功
